This is the frontend webpage site for Social Connect event. Below are the main features of the app.

1. [ ] 2 different types of users: Participants and Admin.
2. [ ] Participants will only be able to access the front/home page, formspage and their own details page.
3. [ ] Participants can view and edit their own response to the form, and their details if need be.
4. [ ] Admin will be able to access additional Data page (Analyse participant data).
5. [ ] Admin is able to edit the page if there is an error.
6. [ ] Admin is able to stop form submission and update pages.

Additional Features
1. [ ] participants to have a swiping mechanism to choose between their favourite profile
2. [ ] Countdown timer at home page for when the application will close
